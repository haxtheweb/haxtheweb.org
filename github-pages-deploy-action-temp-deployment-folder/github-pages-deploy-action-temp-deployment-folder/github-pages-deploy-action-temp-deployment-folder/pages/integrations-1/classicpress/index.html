ClassicPress is a popular fork of WordPress that's even easier to get HAX up and running then it is in the original!

  1. Get the Web component plugin , unzip and rename to webcomponents
  2. Get the HAX plugin , unzip and rename to hax
  3. Put both of these plugins in your wp-content/plugins/ directory
  4. Go into your site's plugins area and enable both wp-admin/plugins.php
  5. Go to edit / create a page/post and you are HAXing
